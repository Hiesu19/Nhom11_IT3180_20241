## Run Locally

Clone the project

```bash
  git clone https://github.com/Hiesu19/Nhom11_IT3180_20241.git
```

Go to the project directory

```bash
  cd Nhom11_IT3180_20241
```

Start the client

```bash
  cd src/client
  npm i
  npm start
```

Start the server

```bash
  cd src/server
  npm i
  npm start
```

**Note:** The `.env` file is required.
